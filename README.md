# Task Manager

A lightweight, browser-based task management application built with vanilla JavaScript. Features include task prioritization, filtering, search functionality, and persistent storage using the browser's LocalStorage API.

## Features

- Create tasks with title, description, priority level, and due date
- Mark tasks as complete or incomplete
- Delete tasks
- Filter tasks by status (All, Completed, Pending, High Priority)
- Search tasks by title or description
- Real-time task statistics
- Data persistence via LocalStorage
- Responsive design for mobile and desktop
- Priority-based color coding

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API


## Usage

1. **Adding a Task**: Fill in the form fields and click "Add Task"
2. **Completing a Task**: Click the checkbox next to the task
3. **Deleting a Task**: Click the delete button on the task card
4. **Filtering**: Use the filter buttons to view specific task categories
5. **Searching**: Use the search bar to find tasks by title or description
